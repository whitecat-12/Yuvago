Informasi Kontroller

Setiap kontroller menyediakan fitur-fitur berikut:

  • Kontrol Real-time: Ubah parameter sensor secara langsung
  • Visualisasi Data: Grafik dan animasi interaktif
  • Ekspor Data: Unduh data pengukuran dalam format PNG/CSV
  • Simulasi Random: Generate data acak untuk pengujian
  • Mode Edukasi: Penjelasan prinsip kerja setiap sensor

Catatan: Kontroller ini dikembangkan untuk tujuan edukasi dalam pembelajaran IoT, embedded systems, dan sensor teknologi.
